# EQT Development Roadmap

### Vision
To become Africa’s most trusted digital marketplace for investment facilitation — connecting credible businesses with growth-ready investors globally.

### Phase 1: MVP (Q4 2025 – Q1 2026)
**Objective:** Build a functional web platform to validate the EQT model.

**Core Deliverables**
- User onboarding (business owners & investors)
- Business listing module with admin verification workflow
- Investor discovery and matchmaking system
- Subscription model (free & paid tiers)
- Secure payment integration (USD transactions)
- Admin dashboard for platform management
- API-ready architecture for mobile extension

**Milestone Targets**
- Alpha release: Internal testing
- Beta release: Invite-only external users
- MVP Launch: Public launch for Nigerian market

---

### Phase 2: Product Expansion (Q2 – Q3 2026)
**Objective:** Expand functionality and strengthen platform scalability.

**Key Additions**
- In-app messaging (with moderation layer)
- Performance analytics dashboard for investors
- Deal documentation templates and secure data room
- Multi-currency wallet integration
- Automated business valuation tool (AI-assisted)

---

### Phase 3: Pan-African Rollout (Q4 2026 – Q2 2027)
**Objective:** Scale EQT to other African markets.

**Targets**
- Regional licensing and compliance framework
- Localization for multiple African markets
- API integration for investment partners and institutions
- Onboarding 1,000+ verified businesses and 500+ investors

---

### Phase 4: Institutional Integration (2027 onward)
**Objective:** Position EQT as the gateway between private investors and institutional capital.

**Initiatives**
- Partnership with investment banks and development agencies
- Launch of EQT Enterprise (institutional investor portal)
- Integration with financial data providers and global deal rooms
